 * Slackbuilds method #2 (sbopkg)
   #sbopkg
   *click sync

   *changelog -see cahnges made
   *packages -see list of installed sbo packages OR remove them
   *updates -update sbo packages
   *browse -search for packages & add to queue
   *search -same as browse but a single package search

   -to install new package
      1) add it to the queue
      2) go to process 
         click ok & then install
         start
      -clear the queue when done

      *tested this with ranger (exit ranger with q like vim)
      -under queue is an option to sort the queue ('sort')
     -can save sbo queues 
     -use 'Load' in queue to load a saved queue
